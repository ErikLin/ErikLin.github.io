Mobile Design System

Pitney Bowes

Overview

Working closely with the design system team, I defined the styling and common patterns that would be used in our iOS and Android apps. The design philosophy was to reimagine our web standards while respecting the style guidelines of each operating system. The first app to leverage this set of modern design system standards was our new flagship app.

Results

  • Modernized the UX/UI for mobile apps
  • Sped up design and development efforts
  • Standardized components across platforms
