Flexbox Sandbox

Advanced Web Design - Spring 2016

Live Site   –   Git Repo   –   View on Behance

Developed as a school project, the primary goal of Flexbox Sandbox is to create a guide for the relatively new flexbox CSS attributes. The site adopts a hands-on approach to learning by allowing users to change the flexbox attributes of an element. This makes playing around with the different properties and values both easier and faster. The supplemental page explains flexbox in a text format in order to solidify the users flexbox education.
